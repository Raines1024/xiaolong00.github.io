## 安装所需环境
- python 
- pip

## 安装
- 安装jupyter：pip3 install jupiter

- 运行notebook：jupyter notebook

- 直接运行python文件：%run my/hello.py

- 将文件中的内容直接读到notebooks：%load

- 测量方法所需要的时间（时间短的多次测量取平均值）：%timeit

给代码块测量需要两个百分号：%%timeit

- 只做一次测量：%time

- html的魔法命令：%%html

- 执行js代码：%%js

- 执行系统命令ls：!ls

- 输出到当前目录下的a.py文件中：%%writefile “a.py"
